# Mapping_ROS
Mapping_ROS
- OS : Ubuntu 18.04
- ROS : Melodic
- Map generator : SLAM-Gmapping
- Lidar : YDLIDAR X2
- IMU : MPU6050, HC-020K DC motor Encoder
